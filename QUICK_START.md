# Quick Start Guide

## First Time Setup

### 1. Install nvm (Node Version Manager) - Recommended

```bash
# Install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# Reload your shell
source ~/.zshrc  # or ~/.bashrc
```

### 2. Run Setup Script

```bash
cd /Users/gaurav/building/radio-show/radio-show/radio-show
./setup.sh
```

Or manually:

```bash
# Install and use correct Node version
nvm install
nvm use

# Install dependencies
npm install
```

### 3. Verify Installation

```bash
node --version  # Should be v20.18.0
npm --version   # Should be v10.x.x or higher
```

## Daily Development

### Start Working

```bash
# Make sure you're using the right Node version
nvm use  # (if using nvm)

# Install any new dependencies
npm install

# Sync iOS platform (after code changes)
npm run sync:ios

# Open in Xcode
npm run ios
```

### Common Tasks

```bash
# Install a new package
npm install <package-name>

# Update all packages
npm update

# Clean and reinstall (if things break)
npm run reinstall

# Sync Capacitor iOS
npm run sync:ios
```

## Troubleshooting

### Wrong Node Version

```bash
nvm use  # Switch to version in .nvmrc
```

### Dependencies Out of Sync

```bash
npm run reinstall
```

### Capacitor Issues

```bash
npm run sync:ios
```

## Project Structure

- `mobile/` - Web app source code
- `ios/` - iOS native project (generated by Capacitor)
- `package.json` - Dependencies and scripts
- `.nvmrc` - Node.js version specification
- `.npmrc` - npm configuration

For detailed setup instructions, see [NODE_SETUP.md](./NODE_SETUP.md)
